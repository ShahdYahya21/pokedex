## Project Overview

This project is a **Pokedex-like REPL (Read–Eval–Print Loop)** built with **TypeScript** that interacts with the **PokeAPI**.

### Features

- Command-line interface for user interaction  
- Fetches real-time Pokémon data from PokeAPI  
- Displays Pokémon details such as name, height, weight, stats, and types  
- Allows catching and listing Pokémon  
- Demonstrates asynchronous programming and API integration in TypeScript
